# LangChain Documentation

This skill provides local reference documentation for LangChain and LangGraph frameworks.

Use this skill when the user asks about:
- LangChain core concepts and architecture
- LangGraph for building stateful multi-agent applications
- Chains and composition patterns
- Prompts and prompt templates
- Memory and conversation history
- Tools and tool calling
- Retrieval and RAG (Retrieval Augmented Generation)
- Vector stores and embeddings
- Document loaders and text splitters
- Agents and agent architectures
- Building LLM-powered applications
